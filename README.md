# Data-Analysis-projects
# Social Media & Marketplace Data Analysis

This repository contains two analytics projects completed during the Finlatics Upskill Program:

- YouTube Trend Analysis
- Facebook Marketplace Analysis

Both projects focus on data cleaning, exploratory data analysis (EDA), visualization, and business insights using Python.

---

# Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# YouTube Trend Analysis

## Objectives
- Analyze trending video categories
- Study audience engagement patterns
- Identify factors affecting video popularity

## Key Insights
- Music and entertainment videos showed high engagement
- Weekend uploads performed better
- Likes and comments strongly influenced popularity

---

# Facebook Marketplace Analysis

## Objectives
- Analyze product listings and pricing trends
- Identify high-performing categories
- Study customer engagement behavior

## Key Insights
- Electronics and furniture categories received higher engagement
- Competitive pricing improved listing visibility
- Customer interaction varied across product categories
